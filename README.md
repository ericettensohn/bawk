#Bawk (Social Post) App

A simple Yik Yak/Twitter clone. Registration, login, password hashing with bcrypt, MySQL.

##Technologies
* HTML
* CSS
* JS
* Angular.js
* SASS
* Flask
* Python
* Jinja
* MySQL

##Local Installation
`pip install flask flask-mysql bcrypt`

`python main.py`
