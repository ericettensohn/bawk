#Bawk (Social Post) App

##Overview
This is a simple Yik Yak/Twitter clone. Registration, login, password hashing with bcrypt, MySQL.

##Technologies
* HTML
* CSS
* JS
* AJAX
* SASS
* Flask
* Python
* Jinja
* MySQL

##Local Installation Requirements
`pip install flask
pip install flask-mysql
pip install bcrypt`
